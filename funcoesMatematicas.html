Funções Matemáticas | Métodos e Propriedades

Métodos para arredondar um número para um inteiro:

  • Math.round(x): Retorna x arredondado para o inteiro mais próximo

  • Math.ceil(x): Retorna x arredondado para o inteiro mais próximo

  • Math.floor(x): Retorna x arredondado para BAIXO o inteiro mais próximo

  • Math.trunc(x): Retorna x arredondado para o inteiro mais próximo

Métodos para verificar se o número é negativo, positivo ou zero

  • Math.sign(x): Retorna se x for negativo, nulo ou positivo

    Se o número for positivo, este método retorna 1

    Se o número for negativo, este método retorna -1

    Se o número for zero, este método retorna 0

Métodos para calcular a elevação de um número

  • Math.pow(x,y): retorna o valor de x elevado a y

Métodos para calcular a raiz de um número

  • Math.sqrt(x): retorna o valor da raiz de x

Métodos retorna um número aleatório entre 0 e 1

  • Math.random(x) : retorna um número aleatório
